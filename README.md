# Let's Punk!

Template project for Punk! omakase web framework for rapid prototyping.

## Setup

```
> brew bundle
> bundle install
```

## Usage

```
> punk db create
> punk db migrate
> punk server
> punk worker
```

## Release Process

Create a new tag and release on GitHub to trigger ship workflow.
